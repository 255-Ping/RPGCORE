# Roadmap & Planning

This folder is a **living working document** — not shipped design specs. Everything here changes as work is done, priorities shift, and new issues are found.

- Items move from sub-pages to `## ✅ Recently Completed` in [todo.md](todo.md) when done.
- Items are added when new issues are discovered.
- Nothing in this folder is a commitment to a timeline.
